# GenAI-Projects

Two AI chatbot projects built as part of my GenAI course.

Project 1 — Zephyr, The Poetic AI Chatbot
Built using Google Gemini API. Responds entirely in poems and lyrical prose using prompt engineering.

Project 2 — English-to-French Translation Chatbot
Built using Hugging Face Transformers (Helsinki-NLP model). Translates English to French with a real-time typing effect.

Tools Used:


Python

Google Colab

Gemini API

Hugging Face Transformers

> Note: These projects are built in Google Colab as part of my learning — not deployed apps, but real working code built from scratch
